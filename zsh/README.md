# zsh

## Terminal settings
-----------------

- iTerm2 (macOS)

Preferences > Profiles > Terminal > Report Terminal Type, set to "xterm-256color".


## Installation

```bash
cp themes/* ~/.oh-my-zsh/themes
ln -sf $(pwd)/zshrc ${HOME}/.zshrc
```

## Use theme

Edit `~/.zshrc`

```bash
ZSH_THEME="af-magic-custom"
```
